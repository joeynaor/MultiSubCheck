# MultiSubCheck
## MultiSubCheck can scan HTTP Headers & subdomain takeover vulnerabilites on all subdomains of a specified domain.

### Requirments:
- Subfinder
- NMAP

### Usage:
```basg
chmod +x MSC.sh
./MSC.sh domain.com
```
